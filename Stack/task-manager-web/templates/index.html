كيف يعمل نظام Stack؟
Push

إضافة مهمة جديدة إلى الأعلى

حذف المهمة من الأعلى

Peek

LIFO

آخر يدخل أول يخرج

مثال عملي:
  1. Push: إضافة "المهمة أ" ← Stack: [أ]
  2. Push: إضافة "المهمة ب" ← Stack: [أ, ب] (ب في الأعلى)
  3. Push: إضافة "المهمة ج" ← Stack: [أ, ب, ج] (ج في الأعلى)
  4. Pop: حذف من الأعلى ← تخرج "المهمة ج" ← Stack: [أ, ب]
  5. Pop: حذف من الأعلى ← تخرج "المهمة ب" ← Stack: [أ]
